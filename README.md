# Portfolio - Candice Carton

Portfolio personnel multilingue (FR/EN) développé en HTML, CSS et JavaScript.

## Contenu

- Page d'accueil
- Compétences
- Cursus
- Projets
- Centres d'intérêt
- Onepager Lore avec visionneuse d'images

## Structure du projet

- [index.html](index.html) : accueil
- [Pages](Pages) : pages internes et scripts
- [Pages/scripts.js](Pages/scripts.js) : logique commune (thème, langue, carrousels)
- [Pages/css](Pages/css) : styles globaux et styles par page
- [images](images) : images du site
- [locales](locales) : traductions i18next

## Lancement en local

Ouvrir [index.html](index.html) dans le navigateur.

Pour un comportement identique à la production (chargement des locales), il est recommandé d'utiliser un petit serveur HTTP local.

- La langue FR/EN est mémorisée dans le navigateur
