# Portfolio — Candice Carton

Portfolio personnel multilingue (FR/EN) développé en HTML, CSS et JavaScript.

## Pages

- Accueil
- Compétences
- Cursus
- Projets
- Centres d'intérêt
- Lore de mon projet de conception de jeu vidéo

## Thème et langue

Le thème clair/sombre ainsi que la langue FR/EN sont mémorisés dans le navigateur.

## Lancement en local

Ouvrir `index.html` dans le navigateur.  
Pour le chargement correct des traductions, l'utilisation d'un serveur HTTP local est recommandée.
